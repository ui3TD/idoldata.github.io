Methods: Extracting Data

Data excluded from Twitter:
  • Posts within the last 7 days
  • Replies
  • Quote retweets
  • Text tweets without media

The rtweet library is used to access the Twitter API. Tweets less than 7 days old are excluded from the analysis to allow time for their numbers to stabilize. It has been observed that replies, quote retweets, and text tweets garner significantly less engagement than simple media tweets. Therefore, the analysis includes only media tweets. Tweets with 0 likes or retweets are given 1 like or retweet in order to avoid undetermined values when taking the logarithm. The number of tweets affected by this intervention is insignificant.

Data excluded from Instagram:

  • Posts within the last 7 days

The instaloader command line tool is used to access the Instagram API. Instagram posts less than 7 days old are excluded from the analysis to allow time for their numbers to stabilize. Posts that disable likes are excluded from the analysis. Posts that disable comments are included in the analysis as 0 comments. This is because it is impossible to distinguish posts with comments disabled from those with comments enabled. A method to exclude posts with statistically low comment to like ratio is being investigated to circumvent this.

The average value of likes, retweets and comments is derived using different methods based on the number of data points in order to avoid overfitting. For greater than 50 data points, LOESS regression is performed. For between 2 and 50 data points, linear regression is performed. For 1 data point, the value itself is taken. The regressions are performed on the base-10 logarithms of the engagement over time. No extrapolation is performed, so if the data date is more recent than the latest post, then the value of the regression at the time of the latest post is taken. The ratio of comments to likes is calculated as the difference of the regressions of the log base-10 of the comments and likes respectively.

Methods: Combined Analysis

Twitter and Instagram stats are combined using Principal Component Analysis (PCA). The PCA is performed to reduce the two dimensions, log of IG likes and log of Twt likes, into one dimension that provides a kind of weighted average (defined by the eigenvector). The average value, called igtwt_avg, is derived from PC1 and scaled to a range of 0 to 1. For individuals with just a Twitter or Instagram account but not both, the missing data is estimated before inclusion in the PCA. Missing data is estimated using the linear regression of the log of IG likes vs. the log of Twt likes.

Glossary

twtlikes: The average number of Twitter likes per media tweet
twtrts: The average number of Twitter retweets per media tweet
twtratio: The ratio of Twitter retweets to likes per media tweet
iglikes: The average number of Instagram likes per post
igcomments: The average number of Instagram comments per post
igratio: The ratio of Instagram comments to likes per post
igtwtratio: The ratio of the log base-10 of Instagram likes to the log base-10 of Twitter likes per post
igtwt_avg: The weighted average of Instagram likes and Twitter likes per post based on PCA
